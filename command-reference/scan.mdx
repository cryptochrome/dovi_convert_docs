## Synopsis

```
dovi_convert -scan [file] [-r depth]
```

**Alias:** `-check`

## Description

Scans MKV files to identify their video format (HDR10, Dolby Vision Profile) and analyzes Profile 7 files to determine FEL complexity. If no file is specified, scans all MKV files in the current directory.

The scan samples 10 timestamps across each file and analyzes the RPU metadata to detect brightness expansion, which indicates Complex FEL.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `[file]` | path | current directory | File or directory to scan. If omitted, scans current directory |
| `-r [depth]` | number | 5 | Scan subdirectories recursively up to specified depth |

## Examples

**Scan current directory:**

```bash
dovi_convert -scan
```

**Scan specific file:**

```bash
dovi_convert -scan "Movie.mkv"
```

**Scan recursively (2 levels deep):**

```bash
dovi_convert -scan -r 2
```

**Scan recursively with default depth (5 levels):**

```bash
dovi_convert -scan -r
```

<Callout icon="circle-info" color="#8caaee" iconType="solid">
Sampling is fast but not exhaustive. A file might have brightness spikes in sections the scan didn't sample. For definitive results, use `-inspect`.
</Callout>

## Output

The scan displays:

- **Video format** - HDR10, Dolby Vision Profile 5/7/8, etc.
- **FEL type** (for Profile 7) - MEL, Simple FEL, or Complex FEL
- **Color-coded verdict** - Green (safe), yellow (verify), red (skip)
